# 釣り人が知っておきたいマリンスポーツの話

釣りに行くと、様々なマリンスポーツをしている人たちと同じ海を共有することになります。

あの人たちは何をしているのか、どういうルールで動いているのか、釣り場としてどう関わってくるのか——知っておくと、トラブルを避けやすくなりますし、海全体への理解も深まります。

釣り人目線で、よく見かけるマリンスポーツを一本ずつまとめました。
