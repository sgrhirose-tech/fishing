# 釣り場で食べるごはんの話

釣りの楽しさを語るとき、ごはんの話はあまり出てきません。でも実際には、何を食べるかで一日の満足度がかなり変わります。

コンビニ飯で済ませることが多い釣り場ごはんを、少し工夫するだけでどれだけ変わるか。バーナーや地元食堂から、釣り場デリバリーというちょっと変わった選択肢まで、店長が2回シリーズで語ります。
