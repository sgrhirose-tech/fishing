# アニメ聖地×釣り場ガイド

## 聖地巡礼しながら竿を出せる場所が、全国にあります

アニメの聖地巡礼と釣り。一見関係なさそうな二つが、場所によってはかなり自然に重なります。

港町が舞台のアニメは多く、主人公たちが海を眺める岸壁や、漁港の風景をそのまま使ったシーンが釣り場でもある——そういう場所が、実は各地にあります。

「釣りだけでは少し物足りない日」「釣りを知らない同行者と来た日」「いつもと違う釣り旅行がしたい日」に、かなり使えるジャンルです。

Tsuricastが調べた「聖地と釣り場が同居する場所」を、店長目線でまとめました。

## 全国のアニメ聖地×釣り場

- [茨城・大洗｜ガールズ＆パンツァーの聖地と大洗港で竿を出す](/articles/column/anime/01_anime_fishing_oarai/)  
  ガルパン一色の商店街と大洗港・大洗磯。サビキ・ちょい投げで楽しめる初心者向けの釣り場です。

- [神奈川・湘南江ノ島｜青ブタ・スラムダンク・ぼざろの聖地で竿を出す](/articles/column/anime/02_anime_fishing_enoshima/)  
  複数の人気アニメの聖地が集中する湘南エリア。聖地として有名な海岸がそのまま釣り場です。

- [静岡・沼津内浦｜ラブライブ！サンシャイン!!の聖地と木負堤防で竿を出す](/articles/column/anime/03_anime_fishing_numazu/)  
  ラブサンの舞台・内浦から近い木負堤防は、アジ・クロダイ・アオリイカが狙える本格堤防です。

- [三重・熊野市｜凪のあすからの舞台で熊野灘の魚を狙う](/articles/column/anime/04_anime_fishing_kumano/)  
  リアス海岸の入り江に広がる聖地と、グレ・アオリイカの本場。わざわざ行く価値のある場所です。

- [和歌山・加太｜サマータイムレンダの舞台で紀淡海峡の魚を狙う](/articles/column/anime/05_anime_fishing_kada/)  
  釣り場として本物の加太港大波止と、廃墟的な雰囲気が残る友ヶ島。関西圏から日帰り可能です。
