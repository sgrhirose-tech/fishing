# 釣り場で食べるごはんの話

釣りの準備をするとき、タックルとエサとウェアは真剣に選ぶのに、ごはんは「コンビニで何か買う」で済ませていませんか。少し工夫するだけで、釣れなかった日の帰り道が変わります。

バーナーやスープジャーで温かいものを用意する話から、釣り場にデリバリーを呼んだ人の話まで、2回シリーズでお伝えします。
