# ベテラン店長の本音アドバイス

![ベテラン店長・田中](/static/img/fishing_master.png)

釣りの技術的な話は、ほかにいくらでもあります。  
このコーナーでは、店に30年立って見えてきた「別の悩み」に答えます。

道具の選び方でも、魚の釣り方でもない。  
**「家族に反対される」「虫エサが無理」「デートで失敗した」「釣具店が怖い」**——  
そういう、検索しても出てこない悩みばかりを集めました。

はっきり言います。釣りをやめてしまう人の多くは、釣り自体が嫌いになったのではありません。  
入口の選び方を間違えたか、周りとの折り合いに失敗しただけです。  
そこを一緒に考えるのが、このコーナーの仕事です。
