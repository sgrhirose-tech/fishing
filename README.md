# シロギス釣り場アドバイザー

投げ釣りで白ギスを釣るのに適した釣り場を、翌日の天気・波・水温をもとに推薦するツールです。

## 対応エリア

- **相模湾**: 辻堂、片瀬西浜（江ノ島）、大磯、平塚、酒匂（小田原）
- **東京湾**: 野島海岸（金沢八景）、富津海岸
- **内房**: 保田海岸、竹岡・関豊海岸
- **外房**: 大原海岸、御宿海岸、守谷海岸（勝浦）

## 使っているデータ

| データ種別 | 提供元 | 費用 |
|-----------|--------|------|
| 風速・風向・降水量 | Open-Meteo API（ECMWFモデル） | **無料** |
| 波高・海面水温 | Open-Meteo Marine API | **無料** |
| AIアドバイス文章（オプション） | Anthropic Claude API | 有料（任意） |

**Open-Meteo のポイント:**
- API キー不要で使えます
- ECMWFの気象モデルを使用しており、約9kmメッシュの精度があります
- アメダスのような陸上観測点ではなく、各釣り場の座標を直接指定して取得するため、海辺の風向き・風速の精度が高いです

## セットアップ手順

### ステップ1: Python のインストール確認

Pythonがインストールされているか確認します。
ターミナル（Mac: Terminal、Windows: コマンドプロンプト）を開いて入力してください:

```
python --version
```

または:

```
python3 --version
```

`Python 3.8.x` 以上と表示されればOKです。
インストールされていない場合は https://www.python.org/downloads/ からインストールしてください。

### ステップ2: ファイルのダウンロード

このリポジトリのファイルを手元のフォルダに保存します。
- `fishing_advisor.py`
- `requirements.txt`

### ステップ3: ライブラリのインストール

ターミナルで、ファイルを保存したフォルダに移動して実行してください:

**Mac/Linux:**
```bash
cd ~/Downloads/fishing   # ファイルを保存したフォルダ（パスは適宜変えてください）
pip3 install requests
```

**Windows:**
```cmd
cd C:\Users\あなたの名前\Downloads\fishing
pip install requests
```

### ステップ4: 実行

```bash
python fishing_advisor.py
```

または:

```bash
python3 fishing_advisor.py
```

翌日の釣り場おすすめレポートが表示され、`fishing_report_日付.txt` というファイルにも保存されます。

### 特定の日付を指定したい場合

```bash
python fishing_advisor.py 2025-06-15
```

## AIアドバイス機能（オプション）

Anthropic Claude APIキーを設定すると、数値データだけでなく、釣り師向けの自然な文章でアドバイスが追加されます。

### APIキーの取得

1. https://console.anthropic.com/ にアクセスしてアカウントを作成
2. 「API Keys」から新しいAPIキーを作成

### 設定方法

**Mac/Linux:**
```bash
export ANTHROPIC_API_KEY=sk-ant-あなたのAPIキー
python fishing_advisor.py
```

毎回入力するのが面倒な場合は、`~/.zshrc` または `~/.bash_profile` に追記します:
```bash
export ANTHROPIC_API_KEY=sk-ant-あなたのAPIキー
```

**Windows（コマンドプロンプト）:**
```cmd
set ANTHROPIC_API_KEY=sk-ant-あなたのAPIキー
python fishing_advisor.py
```

ライブラリも追加でインストールが必要です:
```bash
pip install anthropic
```

## iPhone で使う方法（Pythonista + Shortcuts）

iPhoneから直接レポートを取得できます。**Pythonista 3**（有料アプリ、¥1,220）を使う方法が最も手軽です。

### 必要なもの

- **Pythonista 3**（App Store、¥1,220）: iPhone で Python を動かすアプリ
- Apple **ショートカット**（iOS標準アプリ、無料）: ホーム画面から一発起動するために使う

### ステップ1: Pythonista 3 をインストール

App Store で「Pythonista 3」を検索してインストールします（有料）。

### ステップ2: スクリプトを Pythonista に貼り付ける

1. このリポジトリにある **`fishing_advisor_pythonista.py`** の内容をすべてコピーします
2. Pythonista 3 を開く
3. 右下の「+」ボタン → 「Empty Script」をタップ
4. ファイル名を `fishing_advisor` と入力（拡張子は自動でつきます）
5. コピーしたコードをすべて貼り付けて保存（右上の「Done」）

### ステップ3: Pythonista 内で動作確認

画面上部の「▶」（再生）ボタンをタップして実行します。

- 「気象・海洋データを取得しています...」と表示された後、各釣り場のスコアが出てくればOK
- 最後に「レポートをクリップボードにコピーしました」と表示されます
- メモアプリや LINEに貼り付けて使えます

### ステップ4: Apple ショートカットに登録（ホーム画面から一発起動）

1. **ショートカット**アプリを開く
2. 右上の「+」をタップして新規作成
3. 「アクションを追加」→ 検索欄に「**Pythonista**」と入力
4. 「**Run Script**」（スクリプトを実行）をタップ
5. 「Script」欄で先ほど保存した「**fishing_advisor**」を選ぶ
6. 右上の「∨」をタップしてショートカット名を「釣り場チェック」などに変更
7. 完了

ホーム画面にショートカットを追加すれば、タップするだけでレポートが表示されます。

### 注意事項

- Pythonista 3 は **`requests`** ライブラリを内蔵しているので追加インストール不要です
- **`anthropic`** ライブラリはPythonistaでは使えないため、AIコメント機能は動きません
  - AIコメントを使いたい場合は、将来的なサーバー版またはPC版をご利用ください
- 実行にはインターネット接続が必要です（Wi-Fi または モバイルデータ通信）

## スコアの計算方法

合計100点満点（雨ペナルティで減点あり）

| 項目 | 最大点 | 内容 |
|------|--------|------|
| 底質 | 35点 | 砂地が最高。岩礁・泥地は低スコア |
| 風 | 25点 | 微風・追い風が高スコア |
| 波高 | 20点 | ベタ凪が最高。1m超で0点 |
| 海面水温 | 20点 | 20〜24°Cが最適 |
| 雨ペナルティ | -30点 | 大雨時に減点 |

### 風向きの評価

- **追い風（オフショア）**: 陸から海に向かって吹く風。仕掛けが遠くに飛ぶ。サーファーが来にくいので釣り場が空きやすい
- **向かい風（オンショア）**: 海から陸に向かって吹く風。サーファーが集まりやすい

## よくある質問

**Q: ネットに繋がっていないと動きませんか？**
A: はい、気象データの取得にインターネット接続が必要です。

**Q: 東京湾の波浪データが不正確なことがありますか？**
A: 東京湾は内湾のため、外洋向けの波浪モデルでは精度が落ちることがあります。波の表示はあくまで参考程度にしてください。

**Q: 海しる（海上保安庁）のデータは使えますか？**
A: 現在は各釣り場の底質を固定データとして内蔵しています。将来的に海しるAPIとの連携を追加する予定です。
