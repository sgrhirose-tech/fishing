# 釣法マスタ設計書

**対象チーム:** マスタ作成チーム
**作成日:** 2026-04-01
**ステータス:** 設計確定・データ未作成

---

## 1. 概要

釣法ごとの解説ページ（`/method/{slug}`）実装に向け、
各釣法の説明・難易度・必要な道具・推奨シーズンを管理するマスタデータを新規構築する。

| データ | 内容 |
|--------|------|
| `data/method_master.json` | 釣法ごとの説明・難易度・道具・シーズンマスタ（21釣法） |

> **target_fish（対象魚種）は持たせない。**
> `fish_master.json` の `method` フィールドを逆引きして導出するため、
> `method_master.json` への重複管理は不要。

---

## 2. ファイル構成

```
fishing/
└── data/
    ├── fish_master.json      # 既存: 魚種ごとの季節・釣法・底質マスタ
    └── method_master.json    # 新規: 釣法ごとの説明・難易度・道具マスタ
```

---

## 3. method_master.json の構造

```json
{
  "サビキ釣り": {
    "slug":        "sabiki",
    "difficulty":  1,
    "short_desc":  "カゴにコマセを入れて小魚の群れを寄せる、入門向け定番釣法。",
    "description": "堤防や漁港から行う最も親しみやすい釣法のひとつ。サビキ仕掛けにカゴを取り付け、アミエビなどのコマセを海中に撒いて魚を集める。アジ・イワシ・サバなど回遊魚を手軽に数釣りできる。",
    "gear":        ["サビキ仕掛け", "コマセカゴ", "コマセ（アミエビ）", "磯竿 3〜4m", "スピニングリール"],
    "season":      [3, 4, 5, 6, 7, 8, 9, 10],
    "spot_type":   ["堤防", "漁港"]
  },
  ...
}
```

### フィールド定義

| フィールド | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| `slug` | string | ✓ | URL用スラッグ（英小文字・ハイフン）。ページURL `/method/{slug}` に使用 |
| `difficulty` | int (1〜3) | ✓ | 難易度。1=初心者向け 2=中級者向け 3=上級者向け |
| `short_desc` | string | ✓ | 一覧ページ・OGP用の短い説明（40字以内推奨） |
| `description` | string | ✓ | 詳細ページ本文用の説明（80〜200字程度） |
| `gear` | string[] | ✓ | 最低限必要な道具リスト（3〜6点程度） |
| `season` | int[] | ✓ | 主な釣期の月（1〜12。通年の場合は全月） |
| `spot_type` | string[] | 任意 | 向いている場所の種類（堤防・漁港・磯・サーフ・河川・船 から選択） |

## 5. spot_type の選択肢

`spot_type` に入れる値は以下の文字列から選択する（複数可）。

| 値 | 意味 |
|----|------|
| `堤防` | 防波堤・テトラ帯 |
| `漁港` | 漁港・港内岸壁 |
| `磯` | 自然磯・地磯・沖磯 |
| `サーフ` | 砂浜・砂浜海岸 |
| `河川` | 河口・川 |
| `船` | 乗合船・仕立て船（オフショア） |

---

## 6. 留意事項

- `fish_master.json` の `method` フィールドの文字列と **キー名を完全一致** させること
  （例: `method_master.json` のキーが `"サビキ釣り"` なら `fish_master.json` 側も `"サビキ釣り"`）
- `target_fish` は実装側が `fish_master.json` を逆引きして自動生成するため、**マスタへの記載不要**
- `slug` は一度公開したら原則変更しない（URLが変わると SEO 資産が失われる）
- `difficulty` の基準:
  - 1（初心者）: 専用知識不要・道具が少ない・釣果が出やすい
  - 2（中級）: コツや経験が必要・タックルの選定が重要
  - 3（上級）: 高度な技術・専用タックル必須
